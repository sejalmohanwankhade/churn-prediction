## 🚀 Quick Start

### Option 1: Docker (Recommended)

```bash
# Clone the repo
git clone https://github.com/YOUR_USERNAME/churn-prediction.git
cd churn-prediction

# Start the full stack
docker-compose up --build

# API: http://localhost:8000/docs
# Grafana Dashboard: http://localhost:3000  (admin / admin123)
```

### Option 2: Local Development

```bash
# Install dependencies
pip install -r requirements.txt

# Run the API
uvicorn app.main:app --reload --port 8000

# Run the demo (shows full pipeline output)
python scripts/demo.py

# Run tests
pytest tests/ -v
```

## 🧠 Model Details

### BERT Sentiment Analysis
- Analyses free-text customer reviews to extract sentiment score (−1 to +1)
- Detects high-risk churn phrases (e.g., "thinking about cancelling", "switching to competitor")
- Handles negation: "not great" → negative, not positive
- **Production**: Uses `nlptown/bert-base-multilingual-uncased-sentiment` via HuggingFace Transformers

### Gradient Boosting Churn Classifier
- Input features: tenure, monthly charges, complaints, support calls, contract type, payment method, sentiment score
- Trained on 5,000+ customer records with SMOTE oversampling for class imbalance
- Feature importance: sentiment (30%) > contract type (25%) > complaints (12%)

### Ensemble Fusion
- Final churn probability = GB base probability + BERT risk boost
- Threshold tuned at 0.5 for balanced precision/recall
